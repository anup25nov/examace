🔄 Force Refresh - Clear Cache

This page will help you clear the browser cache and force reload the main application.

Ready to clear cache...

Manual Steps:

  1. Press Ctrl+Shift+R (or Cmd+Shift+R on Mac)
  2. Or open DevTools (F12) → Right-click refresh button → "Empty Cache and Hard Reload"
  3. Or go to Application tab → Storage → Clear storage
