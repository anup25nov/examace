🧪 Test Referral Flow

📱 Test Steps:

  1. Open the app in another tab
  2. Enter a NEW phone number (never used before)
  3. Verify OTP
  4. Check if referral code input appears
  5. Check browser console for debug logs

🔍 Debug Information:

Open browser console (F12) to see debug logs:

  • OTP verification result: - Shows if user is detected as new
  • Is new user: - Boolean indicating new user status
  • User creation time check: - Time-based new user detection
  • Profile creation result: - Database profile creation result

🎯 Expected Behavior:

  • New User: Should see referral code input after OTP verification
  • Existing User: Should go directly to dashboard
  • Debug Button: Should toggle referral input visibility

🐛 Troubleshooting:

If referral input doesn't show for new users:

  • Check console logs for isNewUser value
  • Verify user creation time is recent (< 1 minute)
  • Check if profile creation succeeded
  • Use debug button to force show referral input
